# SmartSales ML - Customer Segmentation using RFM Analysis and K-Means Clustering

## Overview

SmartSales ML is a machine learning project that analyzes retail transaction data to identify customer purchasing behavior and segment customers into meaningful groups.

The project uses:

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* RFM (Recency, Frequency, Monetary) Analysis
* Customer Segmentation
* K-Means Clustering
* PCA Visualization

The goal is to help businesses understand customer value and improve marketing strategies through data-driven customer segmentation.

---

## Dataset

**Dataset:** Online Retail II

The dataset contains transactional records from an online retail store including:

* Invoice Number
* Product Description
* Quantity
* Invoice Date
* Unit Price
* Customer ID
* Country

Files used:

```text
data/raw/
├── online_retail_II_2009_2010.csv
└── online_retail_II_2010_2011.csv
```

## Features

### Data Cleaning

The preprocessing pipeline:

* Removes missing Customer IDs
* Removes cancelled invoices
* Removes transactions with non-positive prices
* Converts dates into datetime format
* Creates a Revenue feature

---

### Exploratory Data Analysis

Analysis includes:

* Total Revenue
* Revenue by Country
* Best-Selling Products
* Daily Sales Trends
* Top Customers by Revenue
* Purchase Frequency Analysis

---

### RFM Analysis

Customer behavior is measured using:

#### Recency (R)

How recently a customer made a purchase.

#### Frequency (F)

How often a customer purchases.

#### Monetary (M)

How much revenue a customer generates.

---

### Customer Segments

Customers are grouped into:

| Segment         | Description                |
| --------------- | -------------------------- |
| Champions       | High-value loyal customers |
| Loyal Customers | Frequent purchasers        |
| New Customers   | Recent customers           |
| At Risk         | Customers likely to churn  |
| Regular         | Average customers          |

---

### K-Means Clustering

The project uses K-Means clustering on:

```text
Recency
Frequency
Monetary
```

Workflow:

1. Feature Scaling
2. Elbow Method
3. K-Means Clustering
4. Cluster Analysis
5. PCA Visualization
